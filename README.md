# 🧠 Editorial Review – Speculative Engine
## Revisione Professionale per SCI-FI · FANTASY · WEIRD

Motore di revisione editoriale avanzato specializzato in narrativa speculativa.

Progettato per:
- racconti
- novelle
- romanzi
- manoscritti in fase di submission

---

# 🎯 Obiettivo

Fornire una revisione:

✓ strutturale  
✓ sistemica  
✓ coerente con il genere  
✓ non invasiva  
✓ orientata alla pubblicazione  

Non è uno strumento di ghostwriting.  
È un motore di analisi editoriale senior.

---

# 🧬 Generi Supportati

## 🚀 SCI-FI
Focus su:
- plausibilità tecnologica
- causalità
- limiti e costi
- conseguenze sistemiche
- failure modes

Evita:
- tecnologia deus ex machina
- infodump scollegati dal conflitto
- incoerenze temporali o logiche

---

## 🐉 FANTASY
Focus su:
- sistema magico coerente
- costi e limiti
- economia e strutture di potere
- mitologia interna

Evita:
- magia risolutiva senza prezzo
- worldbuilding decorativo
- incoerenza politica o geografica

---

## 🜏 WEIRD
Focus su:
- ambiguità controllata
- logica interna invisibile ma coerente
- simboli ricorrenti
- psicologia destabilizzante

Evita:
- confusione senza struttura
- simbolismo puramente ornamentale
- perturbante senza progressione

---

# 🔁 Workflow Interno dell’Agente

1. Caricamento testo (`Read`)
2. Identificazione genere
3. Analisi promessa vs consegna
4. Verifica posta in gioco
5. Mappa tensione narrativa
6. Analisi specifica per genere
7. Prioritizzazione P0 / P1 / P2
8. Tagli consigliati
9. Copyediting tramite pattern (`Grep`)
10. Valutazione + rischio editoriale

---

# 📊 Sistema di Priorità

- **P0** → Bloccante strutturale
- **P1** → Importante ma non fatale
- **P2** → Rifinitura

Regola fondamentale:
Se esistono P0 strutturali → Struttura ≤ 6/10.

---

# 📈 Sistema di Valutazione

Scala:

9–10 → Publish-ready  
7–8 → Solido, polishing necessario  
5–6 → Revisione strutturale richiesta  
3–4 → Problemi gravi  
1–2 → Riscrittura profonda necessaria  

Categorie valutate:

- Struttura
- Personaggi
- Coerenza di Genere
- Stile / Voce
- Chiarezza
- Originalità
- Overall

---

# ⚖ Rischio Editoriale

L’agente valuta anche:

- Rischio commerciale
- Rischio strutturale
- Rischio di leggibilità

Utile in ottica concorsi o submission.

---

# 🧩 Modalità per Testi Lunghi

Se > 10.000 parole:

Analizza:
- Incipit
- Punto medio
- Climax
- Finale

Evita revisione superficiale di tutto il manoscritto.

---

# ✂ Filosofia Anti-Ghostwriting

- Non riscrive intere scene
- Max 500 parole di patch
- Non sostituisce la voce autoriale
- Non standardizza testi già validi

Se testo ≥ 8 → micro-elevazioni, non correzioni invasive.

# 🧭 Principio Fondante

Un testo speculativo non deve solo essere corretto.

Deve:
- sostenere il proprio sistema interno
- mantenere la promessa narrativa
- generare conseguenze credibili
- lasciare una trasformazione

Se non genera frizione, non è pronto.
